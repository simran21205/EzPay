EzPay is a full-stack digital payments platform designed to simplify secure money transfers such as UPI payments and bank transfers. The application provides a smooth user experience with strong backend validation, real-time updates, and secure API handling.

**Features**
UPI Payments

1. Transfer money using sender & receiver UPI IDs
2. PIN verification flow
3. Real-time transaction status (success/failure)
4. Auto-generated unique Transaction ID
================================================================================================

Bank Transfers

1. Transfer funds using account number & IFSC
2. Pre-filled account details for faster transfers
3 .Automatic balance updates
4. Error handling for invalid inputs or insufficient funds
================================================================================================

Beneficiary Management

1. Add new beneficiaries with full details
2. View saved beneficiaries
3. Update & delete beneficiary records
4. Auto-creation of UPI ID for new users
================================================================================================

Transaction History

1. View all previous transactions
2. Includes sender, receiver, amount, date, and status
3. Complete traceability and audit trail
================================================================================================

**Project Goal**

To create a secure, fast, and reliable digital payment system that supports real-time UPI and bank transfers for everyday financial transactions.
